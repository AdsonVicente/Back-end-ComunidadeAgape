⚙️ ComAgape - Back-End (API)
Projeto de Conclusão de Curso (TCC)

Técnico em Desenvolvimento de Sistemas - IDS

Esta é a API robusta que sustenta o ecossistema digital da Comunidade Católica Ágape. Construída com foco em escalabilidade e segurança, ela gerencia o conteúdo histórico, doutrinário e a integração com serviços de Inteligência Artificial.

🛠️ Tecnologias de Elite
Node.js & Express: Ambiente de execução e framework ágil para rotas.

TypeScript: Tipagem estática para um código mais seguro e menos propenso a erros.

Prisma ORM: Manipulação de banco de dados com produtividade e segurança.

Google Gemini API: Integração com IA generativa para auxílio editorial.

JWT (JSON Web Tokens): Autenticação segura de administradores.

Bcrypt: Criptografia de alta segurança para senhas.
